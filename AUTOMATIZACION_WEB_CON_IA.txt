================================================================================
1. ¬øQU√â ES EL DOM?
================================================================================

DOM = Document Object Model (Modelo de Objetos del Documento)

Es la representaci√≥n en forma de √°rbol de todos los elementos de una p√°gina web.
Cuando abres una p√°gina en el navegador, el HTML se convierte en una estructura
de objetos que se puede manipular con JavaScript o herramientas de automatizaci√≥n.

EJEMPLO DE DOM:
---------------
HTML:
    <body>
        <form>
            <input type="text" name="username" />
            <input type="password" name="password" />
            <button type="submit">Login</button>
        </form>
    </body>

DOM (estructura):
    body
    ‚îî‚îÄ‚îÄ form
        ‚îú‚îÄ‚îÄ input[name="username"]
        ‚îú‚îÄ‚îÄ input[name="password"]
        ‚îî‚îÄ‚îÄ button[type="submit"]

¬øPOR QU√â ES IMPORTANTE?
- Puedes "leer" toda la p√°gina program√°ticamente
- Puedes encontrar elementos espec√≠ficos (botones, campos, links)
- Puedes interactuar con ellos (click, escribir, etc.)


================================================================================
2. ¬øQU√â ES LA AUTOMATIZACI√ìN WEB?
================================================================================

Es el proceso de controlar un navegador web mediante c√≥digo para realizar
tareas autom√°ticamente, como si un humano estuviera usando el navegador.

CASOS DE USO COMUNES:
---------------------
‚úì Testing automatizado de aplicaciones web
‚úì Web scraping (extraer datos de sitios web)
‚úì Automatizar tareas repetitivas (llenar formularios, hacer clicks)
‚úì Monitoreo de sitios web
‚úì Generar reportes autom√°ticos
‚úì Bots para redes sociales
‚úì Automatizar compras o reservas

3.1 SELENIUM
------------
La m√°s antigua y establecida. Funciona con m√∫ltiples navegadores.

Instalaci√≥n:
    pip install selenium

Caracter√≠sticas:
- Compatible con Chrome, Firefox, Safari, Edge
- Gran comunidad y documentaci√≥n
- Un poco m√°s lenta que alternativas modernas
- Requiere drivers espec√≠ficos para cada navegador

Ejemplo b√°sico:

    from selenium.webdriver.common.by import By
    
    driver = webdriver.Chrome()
    driver.get("http://127.0.0.1:5000/login")
    
    driver.find_element(By.NAME, "username").send_keys("admin")
    driver.find_element(By.NAME, "password").send_keys("password")
    driver.find_element(By.CSS_SELECTOR, "button[type='submit']").click()
    
    driver.quit()


3.2 PLAYWRIGHT (RECOMENDADO)
-----------------------------
Moderno, r√°pido y potente. Desarrollado por Microsoft.

Instalaci√≥n:
    pip install playwright
    playwright install

Caracter√≠sticas:
- M√°s r√°pido que Selenium
- API moderna y f√°cil de usar
- Soporta Chrome, Firefox, Safari (WebKit)
- Screenshots y videos autom√°ticos
- Mejor manejo de SPAs (Single Page Applications)
- Contextos de navegador aislados

Ejemplo b√°sico:

    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)

        page.fill('input[name="username"]', 'admin')
        page.fill('input[name="password"]', 'password')
        page.click('button[type="submit"]')

4. AUTOMATIZACI√ìN CON INTELIGENCIA ARTIFICIAL
================================================================================

¬øQU√â A√ëADE LA IA?
-----------------
Sin IA: Tienes que programar cada paso manualmente
    ‚Üí "Busca el input con name='username'"
    ‚Üí "Escribe 'admin'"
    ‚Üí "Busca el bot√≥n de submit"
    ‚Üí "Haz click"

Con IA: Le das instrucciones en lenguaje natural
    ‚Üí "Inicia sesi√≥n con usuario 'admin' y contrase√±a 'password123'"
    ‚Üí La IA analiza la p√°gina, encuentra los campos, y ejecuta los pasos

VENTAJAS:
‚úì M√°s flexible - se adapta a cambios en la p√°gina
‚úì M√°s intuitivo - escribes lo que quieres, no c√≥mo hacerlo
‚úì Puede tomar decisiones basadas en el contenido
‚úì Puede manejar casos inesperados

DESVENTAJAS:
‚úó M√°s costoso (uso de APIs de IA)
‚úó Menos predecible que c√≥digo espec√≠fico
‚úó Requiere tokens/cr√©ditos de API
‚úó Posibles errores de interpretaci√≥n


================================================================================
5. C√ìMO FUNCIONAN LOS BOTS WEB CON IA
================================================================================

ARQUITECTURA B√ÅSICA:
--------------------

1. NAVEGADOR CONTROLADO
   ‚Üì
2. EXTRACCI√ìN DEL DOM (HTML de la p√°gina)
   ‚Üì
3. ENV√çO A MODELO DE IA (GPT-4, Claude, etc.)
   ‚Üì
4. IA ANALIZA Y DECIDE QU√â HACER
   ‚Üì
5. RETORNA INSTRUCCIONES (selectores CSS, acciones)
   ‚Üì
6. EJECUCI√ìN DE ACCIONES EN EL NAVEGADOR
   ‚Üì
7. CAPTURA DEL RESULTADO
   ‚Üì
8. (OPCIONAL) RETROALIMENTACI√ìN A LA IA


FLUJO DETALLADO:
----------------

PASO 1: Abrir navegador y cargar p√°gina

    page.goto("http://ejemplo.com")

PASO 2: Extraer el DOM

    # - Solo texto visible: page.inner_text('body')
    # - Estructura accesible: page.accessibility.snapshot()

PASO 3: Crear prompt para la IA
    prompt = f"""
    Analiza esta p√°gina web y encuentra:
    1. El campo de usuario
    2. El campo de contrase√±a
    3. El bot√≥n de login
    
    Retorna los selectores CSS en formato JSON.
    
    HTML:
    {html_content}
    """

PASO 4: Consultar a la IA
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": "Eres un experto en an√°lisis de DOM web"},
            {"role": "user", "content": prompt}
        ]
    )

PASO 5: Parsear respuesta
    result = json.loads(response.choices[0].message.content)
    # Ejemplo: {"username": "#username", "password": "#password", "submit": "button[type='submit']"}

PASO 6: Ejecutar acciones

# archivo: bot_simple.py

from playwright.sync_api import sync_playwright
import time

def automatizar_tareas():
    """Bot simple que crea una tarea en tu aplicaci√≥n."""
    
    with sync_playwright() as p:
        # Configuraci√≥n del navegador
        browser = p.chromium.launch(
            headless=False,  # Ver el navegador
            slow_mo=1000     # Ralentizar para ver qu√© hace
        )
        page = browser.new_page()
        
        try:
            # 1. Ir a la p√°gina de login
            print("üìç Navegando a login...")
            page.goto("http://127.0.0.1:5000/login")
            
            # 2. Llenar formulario de login
            print("‚úçÔ∏è Ingresando credenciales...")
            page.fill('input[name="username"]', 'admin')
            page.fill('input[name="password"]', 'tu_password')
            
            # 3. Click en ingresar
            print("üîë Iniciando sesi√≥n...")
            page.click('button[type="submit"]')
            
            # 4. Esperar a que cargue el dashboard
            page.wait_for_url("**/dashboard")
            print("‚úÖ Sesi√≥n iniciada!")
            
            # 5. Ir a crear tarea
            print("üìù Creando nueva tarea...")
            page.click('a[href="/create_task"]')
            
            # 6. Llenar formulario de tarea
            page.fill('#title', 'Tarea creada por bot')
            page.fill('#description', 'Esta tarea fue creada autom√°ticamente')
            page.select_option('#priority', 'Urgente')
            page.fill('#due_date', '2025-12-31')
            
            # 7. Seleccionar asignados (m√∫ltiple)
            page.select_option('#assignees', ['1', '2'])  # IDs de usuarios
            
            # 8. Enviar formulario
            page.click('button[type="submit"]')
            
            # 9. Verificar √©xito
            page.wait_for_url("**/dashboard")
            print("‚úÖ Tarea creada exitosamente!")
            
            # 10. Screenshot
            page.screenshot(path="tarea_creada.png")
            
            # Esperar un poco antes de cerrar
            time.sleep(3)
            
        except Exception as e:
            print(f"‚ùå Error: {e}")
            page.screenshot(path="error.png")
        
        finally:
            browser.close()

if __name__ == "__main__":
    automatizar_tareas()

# archivo: bot_con_ia.py

from playwright.sync_api import sync_playwright
import openai
import os
import json
from dotenv import load_dotenv

load_dotenv()
openai.api_key = os.getenv("OPENAI_API_KEY")

def analizar_pagina_con_ia(html_content, objetivo):
    """Usa IA para analizar el DOM y decidir qu√© hacer."""
    
    prompt = f"""
    Analiza esta p√°gina web HTML. 
    Objetivo: {objetivo}
    
    Retorna un JSON con los selectores CSS necesarios y las acciones a realizar.
    
    Formato esperado:
    {{
        "acciones": [
            {{"tipo": "fill", "selector": "...", "valor": "..."}},
            {{"tipo": "click", "selector": "..."}}
        ]
    }}
    
    HTML (primeros 3000 caracteres):
    {html_content[:3000]}
    """
    
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": "Eres un experto en automatizaci√≥n web y an√°lisis de DOM."},
            {"role": "user", "content": prompt}
        ],
        temperature=0.3
    )
    
    return json.loads(response.choices[0].message.content)

def ejecutar_acciones(page, acciones):
    """Ejecuta las acciones que la IA determin√≥."""
    
    for accion in acciones:
        tipo = accion['tipo']
        selector = accion['selector']
        
        if tipo == 'fill':
            valor = accion['valor']
            print(f"‚úçÔ∏è Llenando {selector} con '{valor}'")
            page.fill(selector, valor)
            
        elif tipo == 'click':
            print(f"üëÜ Haciendo click en {selector}")
            page.click(selector)
            
        elif tipo == 'select':
            valor = accion['valor']
            print(f"üìã Seleccionando {valor} en {selector}")
            page.select_option(selector, valor)

def bot_inteligente():
    """Bot que usa IA para tomar decisiones."""
    
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=False)
        page = browser.new_page()
        
        # Paso 1: Login con IA
        page.goto("http://127.0.0.1:5000/login")
        html = page.content()
        
        print("ü§ñ Analizando p√°gina de login con IA...")
        acciones = analizar_pagina_con_ia(
            html, 
            "Iniciar sesi√≥n con usuario 'admin' y contrase√±a 'password123'"
        )
        
        ejecutar_acciones(page, acciones['acciones'])
        
        # Esperar navegaci√≥n
        page.wait_for_url("**/dashboard")
        print("‚úÖ Login exitoso!")
        
        # Paso 2: Crear tarea con IA
        page.goto("http://127.0.0.1:5000/create_task")
        html = page.content()
        
        print("ü§ñ Analizando formulario de tarea con IA...")
        acciones = analizar_pagina_con_ia(
            html,
            "Crear una tarea urgente con t√≠tulo 'Revisi√≥n de documentos' para ma√±ana"
        )
        
        ejecutar_acciones(page, acciones['acciones'])
        
        print("‚úÖ Tarea creada!")
        page.screenshot(path="resultado_ia.png")
        
        browser.close()

if __name__ == "__main__":

# archivo: bot_avanzado.py

from playwright.sync_api import sync_playwright
import anthropic
import os
import json
from dotenv import load_dotenv
from typing import Dict, List

load_dotenv()

class BotWebConIA:
    """Bot web inteligente usando Claude de Anthropic."""
    
    def __init__(self, base_url: str):
        self.base_url = base_url
        self.client = anthropic.Anthropic(
            api_key=os.getenv("ANTHROPIC_API_KEY")
        )
        self.browser = None
        self.page = None
    
    def iniciar_navegador(self, headless=False):
        """Inicia el navegador."""
        playwright = sync_playwright().start()
        self.browser = playwright.chromium.launch(headless=headless)
        self.page = self.browser.new_page()
        print("üåê Navegador iniciado")
    
    def cerrar_navegador(self):
        """Cierra el navegador."""
        if self.browser:
            self.browser.close()
            print("üåê Navegador cerrado")
    
    def obtener_dom_simplificado(self) -> str:
        """Obtiene una versi√≥n simplificada del DOM."""
        # Extraer solo lo relevante para reducir tokens
        dom = self.page.evaluate("""
            () => {
                function simplificar(elemento) {
                    const tag = elemento.tagName.toLowerCase();
                    const attrs = {};
                    
                    // Solo atributos relevantes
                    ['id', 'name', 'type', 'class', 'placeholder', 'href'].forEach(attr => {
                        if (elemento.hasAttribute(attr)) {
                            attrs[attr] = elemento.getAttribute(attr);
                        }
                    });
                    
                    return {
                        tag,
                        attrs,
                        text: elemento.innerText?.substring(0, 50) || ''
                    };
                }
                
                // Obtener todos los elementos interactivos
                const elementos = document.querySelectorAll(
                    'input, button, a, select, textarea, form'
                );
                
                return Array.from(elementos).map(simplificar);
            }
        """)
        
        return json.dumps(dom, indent=2)
    
    def preguntar_a_ia(self, objetvo: str, contexto_adicional: str = "") -> Dict:
        """Consulta a Claude para decidir qu√© hacer."""
        
        dom = self.obtener_dom_simplificado()
        url_actual = self.page.url
        
        prompt = f"""
        Est√°s controlando un navegador web.
        
        URL actual: {url_actual}
        Objetivo: {objetvo}
        {contexto_adicional}
        
        DOM simplificado de la p√°gina:
        {dom}
        
        Analiza el DOM y retorna un JSON con las acciones a realizar.
        
        Formato:
        {{
            "razonamiento": "Por qu√© elegiste estas acciones",
            "acciones": [
                {{
                    "tipo": "fill|click|select|navigate",
                    "selector": "selector CSS del elemento",
                    "valor": "valor a ingresar (solo para fill/select)",
                    "descripcion": "Qu√© hace esta acci√≥n"
                }}
            ],
            "confianza": 0.0-1.0
        }}
        
        IMPORTANTE: Genera selectores CSS precisos bas√°ndote en el DOM.
        """
        
        response = self.client.messages.create(
            model="claude-3-5-sonnet-20241022",
            max_tokens=2048,
            temperature=0.1,
            messages=[{
                "role": "user",
                "content": prompt
            }]
        )
        
        resultado = json.loads(response.content[0].text)
        
        print(f"\nüß† IA razon√≥: {resultado['razonamiento']}")
        print(f"üìä Confianza: {resultado['confianza']:.0%}\n")
        
        return resultado
    
    def ejecutar_accion(self, accion: Dict):
        """Ejecuta una acci√≥n individual."""
        tipo = accion['tipo']
        selector = accion.get('selector')
        valor = accion.get('valor')
        desc = accion.get('descripcion', '')
        
        print(f"‚ö° {desc}")
        
        try:
            if tipo == 'fill':
                self.page.fill(selector, valor)
                
            elif tipo == 'click':
                self.page.click(selector)
                
            elif tipo == 'select':
                self.page.select_option(selector, valor)
                
            elif tipo == 'navigate':
                self.page.goto(valor)
            
            # Peque√±a espera para estabilidad
            self.page.wait_for_timeout(500)
            return True
            
        except Exception as e:
            print(f"‚ùå Error ejecutando acci√≥n: {e}")
            return False
    
    def ejecutar_objetivo(self, objetivo: str, contexto: str = ""):
        """Ejecuta un objetivo usando IA."""
        print(f"\nüéØ Objetivo: {objetivo}\n")
        
        # Preguntar a la IA qu√© hacer
        plan = self.preguntar_a_ia(objetivo, contexto)
        
        # Ejecutar cada acci√≥n
        for i, accion in enumerate(plan['acciones'], 1):
            print(f"[{i}/{len(plan['acciones'])}] ", end="")
            exito = self.ejecutar_accion(accion)
            
            if not exito:
                print("‚ö†Ô∏è Fall√≥ una acci√≥n, deteniendo...")
                break
        
        print("\n‚úÖ Objetivo completado!\n")

# Uso del bot
def main():
    bot = BotWebConIA("http://127.0.0.1:5000")
    
    try:
        bot.iniciar_navegador(headless=False)
        
        # Objetivo 1: Login
        bot.page.goto(bot.base_url)
        bot.ejecutar_objetivo(
            "Iniciar sesi√≥n con usuario 'admin' y contrase√±a 'admin123'"
        )
        
        # Esperar navegaci√≥n
        bot.page.wait_for_url("**/dashboard", timeout=5000)
        
        # Objetivo 2: Crear tarea
        bot.ejecutar_objetivo(
            "Ir a crear una nueva tarea con t√≠tulo 'Automatizaci√≥n IA' y prioridad urgente"
        )
        
        # Screenshot final
        bot.page.screenshot(path="bot_resultado.png")
        
        input("Presiona Enter para cerrar...")
        
    finally:
        bot.cerrar_navegador()

if __name__ == "__main__":
    main()
